# Assets Directory

This directory contains all images, icons, and media files for the portfolio website.

## Structure

```
assets/
├── images/
│   ├── profile/          # Profile and avatar images
│   ├── hero/             # Hero/banner images
│   └── projects/         # Project screenshots
├── icons/
│   ├── social/           # Social media icons
│   ├── skills/           # Technology/skill icons
│   └── action/           # Action button icons
├── logos/
│   ├── brand/            # Brand logos
│   └── partners/         # Partner/client logos
└── optimization/
    └── webp/             # WebP optimized versions
```

## Image Optimization Standards

- **Profile Images**: 400x400px, max 100KB (WebP: 50KB)
- **Hero Images**: 1200x600px, max 200KB (WebP: 100KB)
- **Icons**: 64x64px, max 20KB
- **Logos**: 256x256px, max 50KB

## Free Resources Used

- **Icons**: Heroicons, Feather Icons, Material Icons
- **Stock Photos**: Unsplash, Pexels, Pixabay
- **Logos**: Clearbit, LogoSeed
